<tables>

Para defenir uma table usamos as tags <tables></tables>.

Dentro da nossa <tables> vamos usar a tag <tr> que faz uma nova linha na tabela, após fazermos isso temos as tags "th" e "td".  A tag "th" deixa em negrito e coloca as palavras centralizadas já a tag "td" deixa a escrita normal e simplesmente cria uma nova celula.

aqui temos uma base para fazermos uma tabela

            <table>
 
                <tr>

                colspan = "5" DADOS </th>

                </tr>

                    <tr>

                    <th>nome</th>

                    <th>sobrenome</th>

                    <th>idade</th>

                    <th>faculdade</th>

                    <th>data</th>

                </tr>

                <tr>

                    <td>Jeremias</td>

                    <td>Jeremias</td>

                    <td>Jeremias</td>

                    <td>Jeremias</td>

                    <td>Jeremias</td>

                </tr>

            </table>

e assim ira ficar a nossa tabela

agora que aprendemos a fazer uma tabela vamos aprender um pouco de css

css

CSS é como vamos deixar o nosso site mais bonito, com ele podemos estilizar oque fizemos em HTML.

va ate ao head e coloque essas linhas dentro do <style>

.tabela {
    border: black;
    color: black;
    background-color: darkkhaki;
    font-family: Georgia, 'Times New Roman', Times, serif;
}

ao colocar o .tabela estamos criando uma classe dentro do css, sempre que quisermor chamar ela devemos usar o seguinte codigo

<table class="tabela">

observe a mudança que ocore na nossa tabela

lista ordenada e não ordenada

começaremos pela lista ordenada, ao definir uma lista ordenada ela ira enumerar cada idem da nossa lista automaticamente

        
        <ol>
            <li>Jeremias</li>
            <li>Rincaveski</li>
            <li>21</li>
            <li>ESTACIO</li>
        </ol>
  1. Jeremias
  2. Rincaveski
  3. 21
  4. ESTACIO

agora iremos fazer uma não ordenada

<ul>
    <li>Jeremias</li>
    <li>Rincaveski</li>
    <li>21</li>
    <li>ESTACIO</li>
</ul>
  • Jeremias
  • Rincaveski
  • 21
  • ESTACIO

uma pequena resalva que temos que fazer, caso a tag <ol> ou <ul> não forem declarados a lista ira ficar mais a direita fora da caixa invisivel que é a lista e ficara desordenada
